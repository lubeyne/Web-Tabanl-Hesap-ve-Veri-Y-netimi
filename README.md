# Web-Tabanl-Hesap-ve-Veri-Y-netimi
üvenli kullanıcı hesap yönetimi ve web üzerinden dosya transferi sağlayan bir sistem mimarisi kurgulandı.  Kullanıcı verilerinin ve dosyalarının sistem performansını etkilemeyecek şekilde sıkıştırılarak depolanması, arşivlenmesi ve metadata takibi süreçleri üzerinde çalışıldı.
